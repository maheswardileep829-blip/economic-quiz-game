# Economics Quiz Game

A Python-based multiple choice quiz to study economic terms and concepts for DECA competitions.

## Features

- 20 economic terms with definitions
- Multiple choice format (4 options per question)
- Smart wrong answers (uses definitions from other terms)
- Randomized questions each time
- Score tracking and percentage calculation
- Review list of missed terms

## How It Works

The quiz randomly selects economic terms and presents their definitions as multiple choice questions. Wrong answers are pulled from other terms' definitions, making them plausible and challenging.

## Example Terms Covered

- Macroeconomics: GDP, Inflation, Recession, Fiscal Policy, Monetary Policy
- Microeconomics: Supply, Demand, Monopoly, Market Economy
- Finance: Interest Rate, Stock, Dividend, Revenue, Profit
- Business: Asset, Liability, Equity, Capital, Opportunity Cost

## What I Learned

- Dictionary data structures
- Random selection algorithms
- Smart answer generation (pulling from other definitions)
- Multiple choice quiz logic
- User input validation
- Score tracking systems

## Built For

This tool helps me prepare for DECA competitions, specifically:
- Stock Market Game
- Financial Consulting

## Usage
```bash
python economics_quiz.py
```

Choose number of questions (1-20) and answer multiple choice questions!
